一、主要介绍四个win32 api 函数

  1. ShowWindowAsync
  2. SetForegroundWindow
  3. FindWindow
  4. SendMessage

1)、ShowWindowAsync

该函数设置由不同线程产生的窗口的显示状态。

C#形式声明

显示窗体的ShowWindow函数状态的参数——上面cmdshow

2)、SetForegroundWindow

创建指定窗口的线程设置到前台，并且激活该窗口。

C#形式声明

3)、FindWindow

FindWindow这个函数检索处理顶级窗口的类名和窗口名称匹配指定的字符串。这个函数不搜索子窗口。

C#形式声明

4)、SendMessage

该函数将指定的消息发送到一个或多个窗口。此函数为指定的窗口调用窗口程序，直到窗口程序处理完消息再返回。

C#形式声明

wMsg参数常量值
