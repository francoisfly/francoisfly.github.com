使用Hexo和Github创建个人博客

使用Hexo和Github的好处是可以免费的创建个人博客，并且Hexo支持Markdown，使用Markdwon来编写文章，最总可以形成一个静态的html页面。

操作步骤：
1. 安装git
2. 安装node.js
3. 安装hexo
4. 创建仓库在github上
操作环境：

​ windows操作系统

安装git

git官网下载地址 https://git-scm.com/downloads

步骤一：下载并安装git（安装教程）

步骤二：查看安装结果

打开git bash 输入git --version 结果如下:

或者打开cmd 输入git --version

注意：一般出于安全考虑，只有在Git Bash Here中才能进行Git的相关操作。如果需要在cmd命令行里调用Git，那么就要配置电脑的环境变量Path，或者在安装的时候选择use Git from the Windows Command Prompt。

安装node.js

node.js下载地址http://nodejs.cn/download/

步骤一：下载并安装nede.js

步骤二：查看安装结果

打开cmd 输入npm --version或者node --version

安装hexo

步骤一：用cmd跳转到你所需要安装的目录，我是创建在E盘的一个目录中。

步骤二：跳转到指定目录后在命令行中输入npm install -g hexo-cli 开始安装 hexo

步骤三：安装结束后初始化hexo，命令行中输入hexo init

步骤四：首次体验hexo，命令行中输入hexo s 然后在浏览器中输入地址 http://localhost:4000

创建仓库在github上

步骤一：登录github官网,没有帐号的先进注册。

步骤二：新建仓库，仓库名称[Repository name]必须是:账户.github.com，如你登录github的帐号为francoisfy，则仓库名称必须为francoisfy.github.com。

步骤三：将hexo和github关联起来

打开git bash 输入命令git config --global user.email "you@example.com" 和git config --global user.name "Your Name"

步骤四：安装hexo-deployer-git 在命令行输入npm install hexo-deployer-git --save

步骤五：安装结束后输入hexo deploy命令后需要输入github的帐号和密码登录成功后即可上传代码。

步骤六：浏览器其中输入https://francoisfy.github.io（仓库名称：francoisfy.github.com）
